# Aziz Ayachi - Frontend (React)

## Setup Instructions

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```

## Project Structure

- `src/components/Portfolio/` - Portfolio components (converted from HTML)
- `src/components/common/` - Shared components (Cursor, etc.)
- `src/pages/` - Page components
- `src/hooks/` - Custom React hooks
- `src/styles/` - CSS files (your existing styles)

## Features

- ✅ React 18+ with Vite
- ✅ React Router for navigation
- ✅ React Helmet for SEO
- ✅ Lazy loading images
- ✅ All existing CSS preserved
- ✅ Custom cursor functionality
- ✅ Mobile responsive
